# Financial Transactions Analysis Dashboard

A Power BI dashboard built to analyze banking transactions and provide insights into cash flow, spending patterns, account activity, and branch performance.

## Project Overview

This project analyzes over **1,000 financial transactions** to help monitor financial performance through interactive visualizations and key performance indicators (KPIs).

## Dataset

The dataset includes:

- Transaction ID
- Date
- Account Type
- Transaction Category
- Amount
- Description
- Branch

## Project Workflow

1. Data Cleaning using Power Query
2. Data Modeling
3. DAX Measures Creation
4. KPI Development
5. Interactive Dashboard Design
6. Business Insights & Financial Analysis

## KPIs

- Total Cash In
- Total Cash Out
- Net Cash Flow
- Total Transactions
- Average Transaction Amount

## Dashboard Features

- Cash In vs Cash Out Trend
- Transactions by Account Type
- Transactions by Category
- Transactions by Branch
- Top Expense Categories
- Interactive Filters (Date, Account Type, Category)

## Tools Used

- Power BI
- DAX
- Power Query
- Microsoft Excel

## Business Insights

- Monitor cash inflows and outflows.
- Identify the most common transaction types.
- Analyze customer spending behavior.
- Compare account activity.
- Evaluate branch performance.
- Track top expense categories.
